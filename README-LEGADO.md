
# Legado de Astrofotografía — Christian Saavedra

Este repositorio aloja el sitio público de mi obra de astrofotografía. La intención es que permanezca accesible a largo plazo.

## Preservación a largo plazo
- El sitio se publica mediante **GitHub Pages**.
- Se recomienda subir periódicamente una copia del sitio y de las imágenes a **Internet Archive (archive.org)** para preservación. 
  - Captura manual: https://web.archive.org/save/tuusuario.github.io
  - Colección/ítem propio en Archive.org con las imágenes originales.
